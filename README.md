# Ship Skill

A Claude Code skill that commits and pushes your changes in one command.

## What it does

1. Stages all changed files
2. Writes a concise conventional commit message
3. Commits the changes
4. Pushes to the current remote branch
5. Verifies the push succeeded

## Usage

In Claude Code, just type:

```
/ship
```

## Installation

Copy `SKILL.md` into your Claude Code skills directory:

```
~/.claude/skills/ship/SKILL.md
```
